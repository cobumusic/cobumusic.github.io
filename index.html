LinkedIn Helper

A Custom-Built Chrome Plugin

This is a tool I made for myself to make job searching on LinkedIn easier. When installed on Chrome, users can press the "h" key while on the LinkedIn list view for a job search, and all jobs that are either viewed or saved (and hence do not necessarily have to be shown again) will be hidden. I'm currently working on a better UI, bug fixes, and additional filters that aren't built in to LinkedIn's job search. Check it out here!

  • Built with: JavaScript, HTML

Dressertation

An Android App for Cluttered Closets
